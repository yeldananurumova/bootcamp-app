# Password with a very long input

## Preconditions
Signup form is loaded.

## Steps
1. Enter a 1000-character password containing at least one number.
2. Fill the rest of the form validly.
3. Submit.

## Expected Result
The form either accepts it within system limits or rejects it gracefully with a clear error. The app does not crash or hang.

## Severity
Minor
