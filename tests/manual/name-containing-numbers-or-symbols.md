# Name containing numbers or symbols

## Preconditions
Signup form is loaded.

## Steps
1. Enter a name containing a digit or symbol (e.g. "John123" or "John!").
2. Fill the rest of the form validly.
3. Submit.

## Expected Result
The form is rejected with an error stating the name may only contain letters and spaces. No account is created.

## Severity
Major
