# Password one character below minimum (7 characters)

## Preconditions
Signup form is loaded.

## Steps
1. Enter a 7-character password (e.g. "abcdef1").
2. Fill the rest of the form validly.
3. Submit.

## Expected Result
The form is rejected with an error stating the password must be at least 8 characters. No account is created.

## Severity
Minor
