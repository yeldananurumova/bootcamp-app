# Signup with valid username, email, and password

## Preconditions
The signup form is loaded. No account exists yet with the email being used.

## Steps
1. Go to the signup page.
2. Enter a username of 3-20 letters/numbers (e.g. "user123").
3. Enter a unique, valid-format email.
4. Enter a password of at least 8 characters including one number.
5. Submit the form.

## Expected Result
The account is created and the user is signed up successfully.

## Severity
Critical
