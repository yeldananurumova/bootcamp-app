# Signup with all valid fields

## Preconditions
The signup form is loaded. No account exists yet with the username being used.

## Steps
1. Go to the signup page.
2. Enter a username of 3-20 letters/numbers (e.g. "user123").
3. Enter a password of at least 8 characters including one number.
4. Enter a name of letters and spaces only.
5. Enter an age between 13 and 120.
6. Submit the form.

## Expected Result
The account is created and the user is signed up successfully.

## Severity
Critical
