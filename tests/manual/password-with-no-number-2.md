# Password with no number

## Preconditions
Signup form is loaded.

## Steps
1. Enter an 8+ character password made only of letters (e.g. "abcdefgh").
2. Fill the rest of the form validly.
3. Submit.

## Expected Result
The form is rejected with an error stating the password must contain at least one number. No account is created.

## Severity
Major
