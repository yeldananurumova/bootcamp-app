# Password at minimum length with a number (8 characters)

## Preconditions
Signup form is loaded.

## Steps
1. Enter an 8-character password containing at least one number (e.g. "abcdefg1").
2. Fill the rest of the form validly.
3. Submit.

## Expected Result
The password is accepted and the account is created.

## Severity
Minor
